# Youkiz

Web app like [Youtube](youtube.com)

### Getting Started

Install [MongoDb](https://docs.mongodb.com/manual/administration/install-community/)

#### Lunch MongoDb

```
> mongod
```

#### Get project files

```
> git clone https://github.com/ssadiks/youkiz.git
> cd youkiz
> npm install
```

#### Lunch the api

```
> npm run start-server
```

#### Lunch the web app front

```
> npm run start
```
